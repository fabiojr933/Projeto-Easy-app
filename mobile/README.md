# Sistema financeiro mobile

- Faz comunicação com a API 

#### Instale as dependências

```sh
npm install
```

#### Execute a aplicação localmente

```sh
npm start
```

## Tecnologias utilizadas:
- Nodejs
- react-native
- styled-components
- -expo
- -axios
